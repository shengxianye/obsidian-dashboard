# 行为准则 (CODE OF CONDUCT)

## 我们的承诺

为了营造开放、友好的社区环境，我们承诺：

- 对所有背景和身份的人表示欢迎和尊重
- 提供包容和安全的社区环境
- 关注最有利于社区的行为
- 对批评要持开放态度，虚心接受建议

## 我们的标准

**鼓励的行为：**

- 使用友好、包容的语言
- 尊重不同的观点和经验
- 互相帮助和支持
- 提供有建设性的反馈
- 专注于对社区最有利的事情

**不可接受的行为：**

- 骚扰、辱骂、贬低任何人
- 性骚扰或不恰当的行为
- 个人攻击或人身攻击
- 仇恨言论或歧视性言论
- 未经同意发布他人私人信息
- 其他合理认为不适当的行为

## 实施

社区主持人有权利和责任：

- 删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑等
- 暂停参与违反本准则的成员的帐户
- 报告或删除不当内容

## 举报

如果你目睹或经历了不可接受的行为，请通过以下方式举报：

- 提交 GitHub Issue 标记为 `[CODE OF CONDUCT]`
- 在 Discussion 中私下联系项目维护者

所有举报都会被认真对待并进行调查。

## 来源

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org/)，版本 2.1。

感谢所有遵守本准则的社区成员！
