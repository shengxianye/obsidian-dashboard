# 📋 如何添加支付宝收款码

本指南说明如何为项目添加支付宝收款码。

## 第 1 步：生成支付宝收款码

### 方法 1：使用支付宝官方功能（推荐）

1. 打开支付宝 App
2. 首页 → 收付 → **收款**
3. 点击 **生成二维码**
4. 选择 **转账收款** 或 **通用二维码**
5. 点击 **保存图片** 保存到手机

### 方法 2：生成静态二维码

如果你有支付宝账号，访问：
- https://shenghuo.alipay.com/create/qrcode

## 第 2 步：上传图片到项目

1. **将图片放入 `docs/` 文件夹**
   ```
   docs/
   └── alipay-qrcode.png  ← 你的支付宝收款码
   ```

2. **文件命名建议**
   - `alipay-qrcode.png` - 清晰直观
   - `donate-alipay.png` - 也可以
   - 建议使用 PNG 格式，大小 200x200 像素

## 第 3 步：确认 README 中的链接

README.md 中已经添加了这一行：
```markdown
![支付宝收款码](docs/alipay-qrcode.png)
```

如果你用的文件名不同，需要修改这行为正确的文件名。

## 第 4 步：验证显示

1. 在本地使用 Markdown 预览查看
2. 或推送到 GitHub 后在网页版查看
3. 图片应该能正确显示

## 💡 建议

1. **保护隐私** - 确保二维码中不包含敏感个人信息
2. **更新及时** - 如果收款码过期，及时更新
3. **说明清晰** - 明确说明捐赠用途（如维护项目）
4. **表示感谢** - 在 CHANGELOG 中感谢捐赠者（可选）
