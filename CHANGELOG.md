## 如何更新

### 对用户
1. Fork 或克隆最新版本
2. 按 README 安装说明操作
3. 数据文件自动兼容现有内容

### 对贡献者
提交任何改动时，请更新本文件。格式：

```markdown
### [类型]
- 具体改动描述
```

### 类型说明
- **✨ 新增** (Added) - 新功能
- **🔧 改进** (Changed) - 现有功能改进
- **🐛 修复** (Fixed) - Bug 修复
- **🗑️ 移除** (Removed) - 功能移除
- **⚠️ 废弃** (Deprecated) - 即将移除
- **🔒 安全** (Security) - 安全漏洞修复
