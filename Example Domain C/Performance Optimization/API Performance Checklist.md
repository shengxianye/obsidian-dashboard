# 接口性能优化清单

## 数据库层
- [ ] 慢查询分析，添加合适索引
- [ ] 避免 SELECT *，只查需要的字段
- [ ] 大表分页使用游标而非 OFFSET
- [ ] 热点数据加缓存（Redis）

## 应用层
- [ ] N+1 查询合并为批量查询
- [ ] 异步化非关键路径（消息队列）
- [ ] 连接池配置合理（DB / HTTP / Redis）
- [ ] 对象复用，减少 GC 压力

## 网络层
- [ ] 开启 Gzip/Brotli 压缩
- [ ] CDN 加速静态资源
- [ ] HTTP/2 多路复用
- [ ] 接口合并，减少请求次数

## 度量标准
- P50 < 50ms
- P95 < 200ms
- P99 < 500ms
