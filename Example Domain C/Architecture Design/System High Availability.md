# 系统高可用方案

## 可用性目标
| 等级 | 可用率 | 年宕机时间 |
|------|--------|-----------|
| 3个9 | 99.9% | 8.76 小时 |
| 4个9 | 99.99% | 52.6 分钟 |
| 5个9 | 99.999% | 5.26 分钟 |

## 关键手段
### 冗余
- 多机房部署
- 主从数据库 + 自动切换
- 负载均衡 (Nginx / ALB)

### 容错
- 熔断：Hystrix / Sentinel
- 限流：令牌桶 / 滑动窗口
- 降级：非核心功能关闭

### 可观测
- 指标：Prometheus + Grafana
- 日志：ELK Stack
- 链路追踪：Jaeger / SkyWalking
